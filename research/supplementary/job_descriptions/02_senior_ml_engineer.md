# Senior ML Engineer

**Location:** Austin, TX
**Employment Type:** Full time
**Level:** Senior

---

## Summary

The organization seeks a Senior ML Engineer to construct machine learning
systems handling millions of users with sub-100ms latency constraints. The
role encompasses the complete lifecycle from research concepts through
production implementation, including infrastructure design, A/B testing
frameworks, and technical strategy decisions impacting business outcomes.

---

## Key Responsibilities

- Design and implement end-to-end ML pipelines from data ingestion through
  model training, evaluation, and production deployment
- Own model serving infrastructure architecture with performance
  optimization ensuring sub-100ms latency and 99.9% uptime
- Establish experimentation frameworks with A/B testing and statistical
  analysis
- Drive ML infrastructure roadmap decisions and tool evaluation
- Analyze production model behavior and address performance degradation
- Integrate ML predictions into product APIs with backend engineers
- Mentor team members through design and code reviews

---

## Required Qualifications

- 5+ years deploying machine learning systems in production environments
- Deep expertise in Python ML frameworks (PyTorch or TensorFlow)
- Hands-on cloud ML infrastructure experience (AWS SageMaker, GCP Vertex
  AI, or Azure ML)
- Strong system design and distributed computing fundamentals
- Production containerization and orchestration experience (Docker,
  Kubernetes)
- MLOps proficiency (MLflow, Kubeflow, or similar tools)
- SQL skills and data pipeline experience
- Demonstrated ability translating business requirements into technical ML
  solutions

---

## Required Technical Skills

Python, PyTorch, TensorFlow, AWS, Docker, Kubernetes, MLflow, SQL

---

## Preferred Qualifications

- Real-time model serving infrastructure experience with sub-100ms latency
  and high availability
- Production A/B testing track record with rigorous statistical validation
- Infrastructure-as-code familiarity (Terraform, CloudFormation) and GitOps
  workflows
